
## aws-ec2-pflex-volume (beta)
Connect an AWS EC2 instance to PowerFlex on the same VPC and attach a volume.

The file [main.tf](aws-ec2-pflex-volume/main.tf) has the entire sequence of operations and documentation for each step.


## vmware-pflex-datastore
Deploy a VMware Datastore backed by a PowerFlex volume.

For sake of readability, the deployment has been split into two files, one for computing and another for storage.
